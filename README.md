# turntable
随机转盘，不要点炸弹，手持弹幕，抛硬币，随机抽人，随机数

## 大转盘
可以随机选择转盘上的选项，可以自定义转盘选项并保存到本地，切换主题，设置是否可以重复选择等功能。
